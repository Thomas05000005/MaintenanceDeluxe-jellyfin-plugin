# 🧰 MaintenanceDeluxe v0.3.2 — Power-user features + filet de sécurité tests

Trois power-user features qui répondent à des cas d'usage concrets (sauvegarder sa config, partager une URL Uptime Kuma, surveiller le contraste WCAG) + un projet de tests xUnit qui constitue la base de couverture qualité du plugin.

## ✨ Highlights

### 💾 Export / Import config JSON

- 📤 **Bouton Exporter** dans le tab Maintenance qui télécharge la config en JSON.
- 🔒 L'URL du webhook est strippée à l'export par sécurité (placeholder `<redacted>`).
- 📥 **Bouton Importer** ouvre une modal de prévisualisation avec validation.
- 🛡️ L'import touche uniquement le formulaire — le serveur reste intact tant que tu n'as pas cliqué Enregistrer.

### 🩹 Helper Uptime Kuma

- 🔧 Bouton **Construire depuis Uptime Kuma** à côté du champ URL de statut.
- 🛠️ Modal qui demande l'URL de l'instance + slug, génère l'URL finale.
- ✅ Test HTTP HEAD optionnel pour vérifier que l'URL répond.

### 🎨 Validation contraste WCAG

- ⚠️ Warning soft sous le picker de couleur d'accent si le ratio avec le fond est < 4.5:1 (norme AA).
- 📐 Calcul de luminance relative selon la formule WCAG 2.1.
- 🛡️ N'empêche pas la sauvegarde — juste un signal pour éviter les combinaisons illisibles.

### 🧪 Projet de tests xUnit

- 🧪 **61 tests unitaires** sur les fonctions pures (DetectFormat, Normalise*, IsUrlSafe).
- ⚙️ Étape `dotnet test` ajoutée au workflow CI — aucune PR ne peut plus casser une fonction de normalisation sans s'en rendre compte.
