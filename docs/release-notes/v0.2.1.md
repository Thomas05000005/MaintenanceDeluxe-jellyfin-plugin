# 🎨 MaintenanceDeluxe v0.2.1 — Polish Apparence + segmented controls + FR

Polish direct du panneau Apparence introduit en v0.2.0 : correctif d'auth iframe, segmented controls plus modernes, et FR.

## 🐛 Corrigé

- 🔧 L'**iframe live-preview** chargeait avec un **401 Unauthorized** car `preview.html` et `banner.js` exigeaient le header `[Authorize]` — que les browsers n'envoient pas sur une navigation iframe (les Authorization headers ne sont pas portés par les `<iframe src>`).
- ✅ Ces deux endpoints sont désormais **public** (le contenu n'est pas sensible : juste un shell HTML + un script déjà chargé publiquement par JS Injector).

## 🎨 Modifié

- 🎯 Les **3 sélecteurs enum** (Vitesse anim, Densité particules, Bordure carte) passent de plain radio buttons à **pill controls / segmented buttons** — beaucoup plus lisibles et cohérents avec les patterns UI modernes.
- 🌗 Bouton **Réinitialiser apparence** restylé pour être lisible sur les deux thèmes Jellyfin (clair + sombre).
- 🇫🇷 Labels, alertes et copy de l'aperçu **traduits en français** (cohérent avec l'audience initiale francophone).
