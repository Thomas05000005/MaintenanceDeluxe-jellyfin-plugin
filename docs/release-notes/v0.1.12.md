# 🚨 MaintenanceDeluxe v0.1.12 — CRITICAL : bouton Save unresponsive

## 🚨 Hotfix critique

Le bouton Enregistrer ne réagissait plus dès qu'un champ était modifié.

- 🐛 **Root cause** : un `TypeError` dans `collectMessages()` interrompait `initDom` **avant que le handler Save ne soit wiré**. Cause précise : les rows de release notes partageaient la classe CSS `.jf-msg-row` avec les rotation messages, mais elles n'avaient pas de `.jf-msg-detail` correspondant.
- ✅ **Fix** : exclu les `.jf-rn-row` du collecteur rotation, plus un guard `null` défensif dans `collectRowData`.
