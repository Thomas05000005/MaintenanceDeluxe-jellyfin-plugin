# 🖼️ MaintenanceDeluxe v0.1.13 — Fix bordure dorée tronquée avec longues release notes

## 🐛 Corrigé

- 🖼️ La **bordure dorée de la carte était tronquée** quand les release notes faisaient déborder l'overlay au-delà du viewport. Cause : la carte avait son propre `max-height: 92vh` avec `overflow-y: auto` interne — le `::before` (dégradé conique doré) ne s'étendait pas à travers le contenu scrollé.
- ✅ **Fix** : l'overlay scrolle désormais, la carte prend la hauteur full content (la bordure or wrap tout), et les background layers (blobs, beams, grain, particles) sont pinnés au viewport avec `position: fixed`.
