# 🎨 MaintenanceDeluxe v0.2.0 — Personnalisation d'apparence avec live preview

Cette release transforme l'overlay maintenance d'un thème fixe (velours/or) en un **système customisable** où l'admin choisit ses couleurs, sa densité d'animations, son style de bordure. Avec live preview iframe pour itérer sans sauvegarder.

## ✨ Highlights

### 🎨 Nouveau panneau Apparence

- 🎯 **Couleur d'accent unique** : tu choisis 1 couleur, les 5 variantes (soft / bright / deep / shadow + alpha) sont **dérivées automatiquement** pour garder une palette cohérente quel que soit le hue choisi.
- 🌗 **Teinte de fond** customisable (la base de l'aurora background).
- 🎚️ **Opacité de la carte** (40 % à 100 %) — pour laisser plus ou moins respirer l'aurora derrière.
- ⚡ **Vitesse d'animation** : preset `off` / `slow` / `normal` / `fast`.
- ✨ **Densité de particules** : preset `none` / `low` / `normal` / `dense`.
- 🖼️ **Style de bordure carte** : `full` (dégradé conique doré) / `simple` (or plat) / `none`.

### 🔭 Live preview iframe

- 👀 Iframe à côté du formulaire qui rend l'overlay **avec tes settings actuels en temps réel**, sans rien sauvegarder. PostMessage envoyé à chaque champ modifié.
- 🔍 **Bouton "Agrandir l'aperçu"** : passe l'aperçu en plein écran admin.
  - Hotkey `H` pour masquer le panneau settings (focus sur le preview).
  - Hotkey `Esc` pour quitter le mode agrandi.

## 📝 Notes techniques

- 🛡️ Whitelist server-side stricte sur tous les enum (`NormaliseTheme`, `NormaliseAnimationSpeed`, `NormaliseParticleDensity`, `NormaliseBorderStyle`) — toute valeur inconnue retombe sur le défaut, pas de SQL/JS injection possible via la config.
- 🎨 Validation hex stricte `^#[0-9a-fA-F]{6}$` sur les couleurs.
