# 🚨 MaintenanceDeluxe v0.1.11 — CRITICAL : admin UI complètement morte depuis v0.1.8

## 🚨 Hotfix critique

L'admin config page était **entièrement non-responsive depuis v0.1.8**. Plus aucun bouton, plus aucun input, plus aucun save handler ne fonctionnait.

- 🐛 **Root cause** : les templates de release notes contenaient des **vrais retours à la ligne dans des string literals JS** (au lieu de `\n`), ce qui faisait échouer le parsing de tout le script inline `admin`. **Toute la logique JS de la page morte silencieusement** — `<script>` parse error = exécution interrompue sans message d'erreur visible.
- ✅ **Fix** : échappement des newlines dans `RN_TEMPLATES`, vérifié avec `node --check` ajouté au workflow CI pour qu'aucun futur commit ne puisse réintroduire cette classe de bug.

> 💡 Cette release a inspiré la grosse refacto v0.3.7 qui a extrait tout l'inline JS dans `admin.js`, fermant définitivement la classe de bug.
