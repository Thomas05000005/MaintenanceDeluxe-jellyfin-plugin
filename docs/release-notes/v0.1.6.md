# 🔑 MaintenanceDeluxe v0.1.6 — Admin peut atteindre la page login pendant la maintenance

## ✨ Ajouté

- 🔑 Le bouton **dismiss** était caché sur la page de login (puisque l'utilisateur n'était pas encore authentifié). Problème : si l'admin se déconnectait par accident, il était **bloqué** par son propre overlay sans pouvoir se reconnecter.
- ✅ **Fix** : le bouton *Accès administrateur →* est désormais **toujours visible** sur la page login (même icon que le bouton admin dismiss). Cliquer dessus retire l'overlay pour la session courante, l'admin peut taper ses identifiants et se reconnecter.
- 🛡️ Sécurité : les non-admin qui dismissent et tentent de se logger sont rejetés par le backend (compte disabled) — pas de bypass possible.
