# 🎬 MaintenanceDeluxe v0.1.0 — Initial MVP release

Première release publique du plugin. Overlay maintenance plein-écran avec compte à rebours, release notes admin-configurables, theming velours/or, et désactivation automatique des non-admin via l'API Jellyfin.

## ✨ Highlights

### 🎬 Overlay maintenance

- 🎞️ **Plein écran** sur l'UI Jellyfin web — quand la maintenance est active, les utilisateurs non-admin voient ton overlay au lieu de Jellyfin.
- ⏱️ **Compte à rebours adaptatif** :
  - Heure cible absolue (ex. *Retour prévu à 21h30*)
  - Hint relatif (ex. *≈ 25 min*)
  - Minutes + secondes précises sous 5 min restantes
- 📊 **Barre de progression time-based** pilotée par les start/end planifiés. Pulse en orange si tu dépasses la fenêtre prévue.
- 📝 **Release notes admin-configurables** : titre custom, sous-titre, autant de sections que tu veux avec emoji + titre + body markdown.

### 🎨 Theming cinema-velours

- 🌌 Palette velours (rouge sombre + or + minuit) avec aurora background dérivant lentement.
- 🎞️ Logo film reel rotatif au centre.
- ✨ Particules embers crépitantes (densité auto-adaptée selon la perf machine).
- 🖼️ Bordure dorée brushed-metal autour de la carte.

### ⚡ Performance tiers auto-détectés

- 💻 `full` (desktop + mobile modernes) : tous les effets
- 📺 `reduced` (Tizen / webOS / Fire TV / Android TV) : pas de particules, background statique
- ♿ `minimal` (`prefers-reduced-motion: reduce`) : background uni, zéro animation

### 🔧 Backend

- 🗓️ **Programmation horaire** : activation auto / désactivation auto / redémarrage serveur.
- 👤 **Désactivation automatique** des comptes non-admin via `IUserManager.UpdatePolicyAsync` — les utilisateurs voient une "account disabled" sur les apps natives qui ne supportent pas l'overlay JS.
- 🔒 **Tracking** des UUIDs disabled vs pre-disabled pour préserver l'intent admin à la désactivation.

## 📦 Prérequis

- Jellyfin **10.11.6** ou plus récent.
- 2 plugins prérequis : **JavaScript Injector** + **File Transformation**.
