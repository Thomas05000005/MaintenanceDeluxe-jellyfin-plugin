# 🔍 MaintenanceDeluxe v0.3.6 — Aperçu admin agrandi : la carte remplit la zone

Une amélioration UX **strictement admin-side** : quand tu cliques *Agrandir l'aperçu* dans le panneau Apparence, la carte de prévisualisation s'élargit pour occuper la zone visible au lieu de rester en mode compact et de paraître perdue dans le fond aurora.

## ✨ Highlights

### 🔍 Preview admin élargi

- 📏 La carte de prévisualisation passe de **640 px (compact) à `min(1400px, 92vw)`** quand le preview est en mode agrandi.
- 🛡️ **Zéro impact côté utilisateurs finaux** : l'élargissement est appliqué via une classe CSS `jf-md-preview-expanded` activée **uniquement** par un postMessage du parent admin vers l'iframe. Les sessions Jellyfin réelles ne reçoivent jamais ce message.
- 🎬 Concrètement : tu peux maintenant **évaluer correctement les détails de ta card** (release notes, comparaisons, accent color) en mode preview agrandi, sans avoir besoin d'activer la maintenance pour de vrai pour vérifier le rendu.

## 📝 Notes techniques

- 🔧 `configPage.html` : nouvelle fonction `notifyPreviewExpanded(bool)` qui envoie un `postMessage({ type: 'md-preview-expanded', expanded })` à l'iframe à chaque `enterExpand()` / `exitExpand()`.
- 🔧 `banner.js` : le handler `message` du mode live-preview reconnaît `md-preview-expanded` et add/remove la classe CSS. Same-origin guard + `event.source === parent` toujours actifs.
- 🛡️ CSS : `#jf-md-overlay.jf-md-preview-expanded .jf-md-card { max-width: min(1400px, 92vw) !important; }` — le clamp 92vw empêche tout débordement même sur petit écran admin.
