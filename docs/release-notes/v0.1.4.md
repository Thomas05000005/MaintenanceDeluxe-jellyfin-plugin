# 🛠️ MaintenanceDeluxe v0.1.4 — SPA nav watchers + MutationObserver + preview sessionStorage

## ✨ Ajouté

- 🔁 **SPA nav watchers** : `hashchange`, `popstate`, `pushState/replaceState` wrappés — l'overlay réagit à toutes les navigations Jellyfin, pas seulement au reload page.
- 🛡️ **MutationObserver** sur `document.body` : si React de Jellyfin re-render et fait disparaître notre overlay, on la **réinjecte automatiquement** dans la milliseconde.
- 💾 **Preview sessionStorage cache** : le flag `?md-preview=1` est mis en cache après la première détection, car Jellyfin HashRouter peut strip les query strings au boot.

> ⚠️ Cette release contenait une syntax error qui cassait tout. Voir le hotfix v0.1.5.
