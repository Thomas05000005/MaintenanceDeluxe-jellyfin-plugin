# 🩹 MaintenanceDeluxe v0.1.2 — Fix Preview + overlay sur page login

## ✨ Ajouté

- 👀 Le **bouton Preview** dans l'admin ouvre maintenant un nouvel onglet avec `?md-preview=1` qui rend l'overlay avec les vraies settings, sans activer la maintenance pour de vrai.
- 🔓 L'overlay s'affiche désormais **aussi sur la page login** (`/login.html`) — les utilisateurs disabled voient le message custom au lieu d'une erreur "compte désactivé" brutale.
- 🌐 Pour permettre ça, le endpoint `GET /MaintenanceDeluxe/maintenance` est désormais **public** (pas d'`[Authorize]`).
