# 🇫🇷 MaintenanceDeluxe v0.1.3 — Fix accents français

## 🐛 Corrigé

- 🇫🇷 Les **accents français** dans l'overlay (titre, sous-titre, release notes : é, à, ê, œ, ç) s'affichaient comme mojibake sur certaines configurations Jellyfin.
- ✅ **Root cause** : JS Injector sert le script sans charset UTF-8 explicite, le browser interprète en Latin-1. **Fix** : tous les caractères non-ASCII sont désormais **escaped en `\uXXXX`** dans `banner.js` — rendu identique quel que soit le charset du serveur.
- ⚙️ Script `scripts/escape_non_ascii.py` ajouté au CI pour catch toute régression.
