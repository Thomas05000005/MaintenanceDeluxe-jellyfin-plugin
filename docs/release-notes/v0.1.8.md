# 💾 MaintenanceDeluxe v0.1.8 — Save résilient + autosave preview + templates RN

## ✨ Ajouté

- 💾 **Save résilient** : si l'admin clique Enregistrer et que la requête échoue, le formulaire reste rempli avec ses modifications (au lieu de se vider). Retry possible sans tout reretaper.
- 💾 **Autosave de l'aperçu** : l'iframe live-preview se met à jour automatiquement à chaque modification de champ (input, change), sans bouton Refresh.
- 📋 **Templates de release notes** : 12 templates pré-faits (Update mineure, Update majeure, Migration, Backup, etc.) accessibles via dropdown — clique pour pré-remplir titre + body + icône, puis personnalise.
