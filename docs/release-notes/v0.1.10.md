# 🛠️ MaintenanceDeluxe v0.1.10 — Pages admin toujours sans overlay (race fix)

## 🐛 Corrigé

- 🏎️ Race condition : l'overlay maintenance pouvait **s'afficher brièvement sur une page admin** avant que le check `IS_ADMIN` (qui dépend de la fetch async `getCurrentUser`) se termine. Concrètement : tu te connectes en admin, l'overlay flashe pendant 200-500 ms, puis disparaît.
- ✅ **Fix** : drop du guard `IS_ADMIN` autour de `isAdminPage()` — Jellyfin redirige déjà les anonymes loin des routes `/dashboard/*` etc., donc si on est sur une admin route on est forcément admin. Le check synchrone suffit, plus de race.
