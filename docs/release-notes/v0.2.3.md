# ✨ MaintenanceDeluxe v0.2.3 — Bordure rotative + polish bouton expand

Deux retouches sur le nouveau panneau Apparence introduit en v0.2.0.

## ✨ Ajouté

- 🌀 **Bordure de carte "Rotative"** : un dégradé doré qui tourne lentement autour de la carte de maintenance, avec un halo pulsant léger. La vitesse de rotation suit le curseur d'animation (incluant le mode `off` qui fige tout). Effet utilisé sur les annonces très importantes — donne une présence visuelle marquée sans être agressif.

## 🐛 Corrigé

- 🎨 Le bouton **Agrandir l'aperçu** utilisait l'`emby-button` Jellyfin qui rendait avec un **fond blanc en thème sombre** — illisible. Il utilise désormais le même style neutre-translucide que le bouton *Réinitialiser apparence* pour s'intégrer sur les deux thèmes (clair + sombre).
