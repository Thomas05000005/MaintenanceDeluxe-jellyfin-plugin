# 🛠️ MaintenanceDeluxe v0.3.0 — Quality release : bug fixes + anti-data-loss + UI française

Première vraie release qualité du plugin. Trois axes : correctifs robustesse, mécanismes anti-perte de données dans l'admin UI, et traduction française complète.

## 🐛 Corrigé

- 🛡️ `BannerController` retourne désormais **503 Service Unavailable** quand `Plugin.Instance` n'est pas encore initialisé.
- 🔧 `ScheduledRestart` est préservé quand il est planifié après la fenêtre de maintenance.
- 🧹 `MutationObserver` et `hideScrollObserver` détachés au `pagehide` — plus de fuite mémoire après une longue session.
- 🔒 Le receiver `postMessage` du live-preview impose un check **same-origin** (anti-injection cross-origin).

## ✨ Anti-perte de données admin UI

- 🔴 **Badge orange ●** à côté du bouton Enregistrer dès qu'il y a des modifs non sauvegardées.
- ⚠️ **Browser warning** (`beforeunload`) si tu fermes l'onglet avec des changements pending.
- 💾 **Autosave localStorage** toutes les 500 ms + prompt de restauration au reload.
- ✅ Validation inline : `scheduledEnd < scheduledStart`, `scheduledRestart` dans le passé.

### 🎨 Live-preview robuste

- 🔄 Fallback texte clair après 5 s si l'iframe n'a jamais signalé ready (au lieu d'un écran blanc).

## 🇫🇷 UI admin française

- 🇫🇷 Tab Maintenance, bouton Enregistrer, modale de confirmation, alertes et validations **entièrement traduits**.
