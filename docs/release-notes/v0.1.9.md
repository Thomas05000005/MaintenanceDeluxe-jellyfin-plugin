# 🛡️ MaintenanceDeluxe v0.1.9 — Pas d'overlay sur pages admin + persist dismiss

## ✨ Ajouté

- 🛡️ L'overlay **ne s'affiche plus sur les pages admin** (`#!/dashboard/*`, `#!/users`, `#!/configurationpage`, etc.) — l'admin a toujours accès à son outil de gestion même quand la maintenance est active. Si tu cliques *Accès admin* depuis l'overlay, tu peux maintenant naviguer librement dans la dashboard sans que l'overlay revienne.
- 💾 **Persist dismiss en sessionStorage** : si l'admin ferme l'overlay via *Accès admin*, son choix survit aux remounts React de Jellyfin (sans persister entre tabs / sessions).
