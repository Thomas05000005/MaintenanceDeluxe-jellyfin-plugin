# 🎚️ MaintenanceDeluxe v0.2.2 — Inputs numériques pour fine-tuning + polish layout

Aller au-delà des presets : chaque preset *Vitesse d'animation* et *Densité de particules* dispose maintenant d'un input numérique compagnon pour overrider la valeur du preset.

## ✨ Ajouté

- 🎚️ **Inputs numériques de précision** à côté des presets :
  - Vitesse d'animation : **0 à 5×** (0 = animations désactivées, 1 = normal, 0.5 = 2× plus rapide, 2 = 2× plus lent)
  - Densité de particules : **0 à 500** particules
- 💡 **Comportement intuitif** : cliquer un preset remplit le numérique (valeur de référence). Taper un numérique custom override le preset choisi.

## 🎨 Modifié layout panneau Apparence

- 📌 L'aperçu d'apparence est maintenant **sticky** — il suit ton scroll dans la colonne contrôles, plus besoin de remonter pour voir l'effet d'un slider.
- 📐 Iframe min-height bumpé à **540 px** pour que les longues release notes tiennent à l'écran.
- ✂️ Colonne contrôles légèrement plus compacte ; en mode *Agrandir l'aperçu* le panneau settings réduit de 340 px → 300 px (10 % de plus d'espace pour le preview).
